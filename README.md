#  Student Grade Management System (Java)

A **console-based Java application** designed to help teachers manage student grades efficiently.  
The system allows adding student records, storing them using Java Collections, and generating a sorted grade report.

---

##  Features
- Add student names and grades
- Store data using `HashMap` (Student Name ‚Üí Grade)
- Sort students based on grades
- Display all student records
- Menu-driven console interface
